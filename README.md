# Home Project

Структура проекта для ремонта квартиры, хранения материалов и ведения рабочих решений.

## Структура

- `docs/` — цели проекта и общие документы.
- `plans/` — планы квартиры, обмеры и схемы расстановки.
- `renders/` — интерьерные визуализации по помещениям.
- `budget/` — бюджет, сметы и фактические расходы.
- `decisions/` — журнал решений и проверки совместимости.
- `smart/` — заметки по умному дому, автоматизации и сценариям.

## Текущие материалы

- `docs/project-goal.md` — цель проекта и критерии контроля.
- `plans/apartment-measurements.pdf` — обмерный план квартиры.
- `plans/furniture-layout.pdf` — план расстановки мебели.
- `renders/living-room/` — рендеры гостиной и кухни.
- `renders/bedroom/` — рендеры спальни и рабочего места.

## Как использовать

- Все новые планы и технические схемы складывать в `plans/`.
- Все визуализации и референсы по помещениям складывать в `renders/`.
- Все расходы фиксировать в `budget/budget.md`.
- Все принятые решения и спорные варианты фиксировать в `decisions/decision-log.md`.
- Перед закупкой и монтажом проверять связки решений в `decisions/compatibility-checks.md`.
