# Проверка совместимости решений

## Цель

Проверять решения до заказа и монтажа, чтобы исключать конфликты по размерам, подключению, эстетике и бюджету.

## Чеклист

| Зона | Что проверяем | Статус | Риск | Комментарий |
| --- | --- | --- | --- | --- |
| Кухня | Размеры мебели и техники |  |  |  |
| Кухня | Электрика и точки подключения |  |  |  |
| Санузел | Сантехника, инсталляции, размеры |  |  |  |
| Спальня | Кровать, проходы, тумбы, свет |  |  |  |
| Гостиная | ТВ-зона, розетки, мебель, свет |  |  |  |
| Вся квартира | Цвета, материалы и стилистика |  |  |  |
| Вся квартира | Влияние на бюджет и сроки |  |  |  |

## Критерии проверки

- Размеры совпадают с планом и реальными условиями.
- Решение не блокирует соседние элементы.
- Есть все нужные выводы, розетки и подключения.
- Материалы и цветовые сочетания не конфликтуют.
- Изменение укладывается в бюджет и сроки.
